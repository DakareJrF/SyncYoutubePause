# YouTube Pause on Multiple Tabs

Пользовательский скрипт для **Tampermonkey** / **Greasemonkey**, который  
автоматически **ставит на паузу** видео на **YouTube**, если в данный момент  
открыто **две и более вкладок** с сайтом `youtube.com`.

---

## Как работает

1. Каждая вкладка «сигнализирует» о себе, записывая в `localStorage`  
   метку времени (`yt_tab_<id>`) каждые 2 секунды.
2. Если активных вкладок больше одной, скрипт:
   - синхронизирует состояние «play / pause» между всеми вкладками;  
   - останавливает воспроизведение во всех вкладках, если хотя бы в одной  
     пользователь нажал «паузу».
3. При закрытии вкладки её метка удаляется автоматически.

---

## Установка

1. Установите расширение [**Tampermonkey**](https://www.tampermonkey.net/)  
   (Chrome, Edge, Firefox, Opera, Safari).
2. Откройте [сырую ссылку на скрипт](YouTube Pause on Multiple Tabs-1.0.user.js)  
   → Tampermonkey предложит установить. Нажмите **«Установить»**.
3. Готово! Перейдите на YouTube и откройте несколько вкладок с видео,  
   чтобы проверить работу.

---

## Настройки

В начале скрипта можно изменить две константы:

| Константа           | По умолчанию | Описание |
|---------------------|--------------|----------|
| `TAB_LIFETIME`      | 5000 мс      | Через сколько мс считать вкладку «мёртвой», если нет «сигнала». |
| `HEARTBEAT_INTERVAL`| 2000 мс      | Как часто вкладка «пишет» о себе в `localStorage`. |

---

## Совместимость

- **Браузеры:** любой современный Chrome, Firefox, Edge, Safari.  
- **Сайт:** только `https://www.youtube.com/*`.  
- **Дополнительные разрешения:** не требуются (`@grant none`).
